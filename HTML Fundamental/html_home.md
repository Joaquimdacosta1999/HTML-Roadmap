# Learn HTML

HTML is the standard markup language for creating web pages.
With HTML you can create your own Website. This tutorial teaches you everything about HTML.
HTML is easy to learn - You will enjoy it!

## Learning by Examples

```html
</head>

</body>
</html>
```

## HTML Exercises

<details>
<summary><b>Question 1</b></summary>

**What is a correct syntax for an HTML hyperlink?**

* `Option 1:` `<a href='/home.htm'>Visit W3Schools.com!</a>`
* `Option 2:` `<link href='/home.htm'>Visit W3Schools.com!</link>`
* `Option 3:` `<alink href='/home.htm'>Visit W3Schools.com!</alink>`

<br>

> **Answer:** Option 1 is correct because the `<a>` tag is the standard element for hyperlinks.
</details>
