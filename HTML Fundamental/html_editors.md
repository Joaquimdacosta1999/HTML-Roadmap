# Learn HTML Using Notepad or TextEdit

Web pages can be created and modified by using professional HTML editors.

However, for learning HTML we recommend a simple text editor like Notepad (PC) or TextEdit (Mac).

We believe that using a simple text editor is a good way to learn HTML.

Follow the steps below to create your first web page with Notepad or TextEdit.

## Step 1: Open Notepad (PC)

<b>Windowns 8 or later:</b>

Open the <b>Start Screen</b> (the window symbol at the bottom left on your screen). Type <b>Notepad</b>.


<b>Windows 7 or earlier:</b>

Open <b>Start</b> > <b>Programs</b> > <b>Accessories</b> > <b>Notepad</b>


## Step 1: Open TextEdit (Mac)

Open <b>Finder</b> > <b>Applications</b> > <b>TextEdit</b>

Also change some preferences to get the application to save files correctly. In <b>Preferences</b> > <b>Format</b> > choose <b>"Plain Text"</b>

Then under "Open and Save", check the box that says "Display HTML files as HTML code instead of formatted text".

<b>Then open a new document to place the code.</b>

## Step 2: Write Some HTML

Write or copy the following HTML code into Notepad:

``` html
<!DOCTYPE html>

<html>

<head>

<title> First Page</title> 

</head>

<body>

<h1>My First Heading</h1>

<p>My first paragraph.</p>

</body>

</html>
```

## Step 2: Save the HTML Page
Save the file on your computer. Select <b>File</b> > <b>Save</b> as in the Notepad menu.

Name the file <b>"index.htm"</b> and set the encoding to <b>UTF-8</b> (which is the preferred encoding for HTML files).

Open the saved HTML file in your favorite browser (double click on the file, or right-click - and choose "Open with").
