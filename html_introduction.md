# HTML Introduction

HTML, or HyperText Markup Language, is the standard language used to create web pages. It provides the basic structure for web documents and is enhanced and styled by other technologies like CSS (Cascading Style Sheets) and JavaScript.

## Key Features:
- **Structure**: HTML uses a series of elements and tags to define the structure of web pages.
- **Elements & Tags**: An HTML document is made up of elements, which are represented by tags that typically have an opening and a closing tag (e.g., `<tagname>content</tagname>`).
- **Attributes**: Tags can have attributes that provide additional information about the element. For example, `<a href="https://example.com">Link</a>` has an `href` attribute.

## Basic Structure of an HTML Document:
```html
<!DOCTYPE html>
<html>
<head>
    <title>Your Title</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is a paragraph.</p>
</body>
</html>
```

## Conclusion:
HTML is an essential technology for web development. It allows for the creation of structured documents that can be consistently rendered across different browsers and devices.
